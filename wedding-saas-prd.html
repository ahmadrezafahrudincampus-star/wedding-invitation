01 — Product Overview

Mengubah undangan
menjadi pengalaman

Velvet Vows adalah platform SaaS yang memungkinkan calon mempelai membuat undangan pernikahan digital yang elegan, interaktif, dan fully customizable — tanpa menyentuh satu baris kode pun. Admin dan mempelai mengakses CMS visual yang intuitif untuk mengatur konten, tema, animasi, dan fitur sosial.

⬡

No-Code Builder

Antarmuka drag-and-drop visual yang memungkinkan kustomisasi penuh tanpa keahlian teknis. Setiap perubahan di-preview secara real-time.

CMS Core
◈

Multi-Tema Dinamis

Library tema premium yang mencakup Golden Blossom, Rustic Wood, Minimalist Modern dan lainnya — masing-masing dengan palet, font, dan ornamen unik.

Visual Engine
◎

SPA yang Performa

Output undangan berupa Single Page Application teroptimasi: lazy loading, kompresi gambar otomatis, animasi CSS ringan dengan skor Lighthouse 90+.

Performance
⬡

Guest Personalization

Sapaan tamu yang dipersonalisasi via parameter URL (?to=Nama+Tamu) sehingga setiap undangan terasa eksklusif dan personal.

UX Feature
◈

Supabase Powered

Backend serverless menggunakan Supabase untuk database PostgreSQL, storage file, auth, dan real-time subscription untuk buku tamu live.

Backend
◎

RSVP & Analytics

Sistem RSVP terintegrasi dengan dashboard konfirmasi tamu, tracking kehadiran, dan analitik kunjungan undangan real-time.

& Estetika

07 — State Management & Implementation

Rendering Config
ke Komponen

hooks/useInvitationConfig.js 
import { useState, useEffect } from 'react'
import { supabase } from '@/lib/supabase'

export function useInvitationConfig(slug) {
  const [config, setConfig] = useState(null)
  const [loading, setLoading] = useState(true)
  const [error, setError] = useState(null)

  useEffect(() => {
    async function fetchConfig() {
      const { data, error } = await supabase
        .from('invitations')
        .select('config')
        .eq('slug', slug)
        .eq('is_published', true)
        .single()

      if (error) { setError(error); return }
      setConfig(data.config)
      applyTheme(data.config.visual_configs)
      setLoading(false)
    }
    fetchConfig()
  }, [slug])

  return { config, loading, error }
}

// Inject CSS variables dari theme config ke :root
function applyTheme({ primary_color, secondary_color, font_heading, font_body }) {
  const root = document.documentElement
  root.style.setProperty('--color-primary', primary_color)
  root.style.setProperty('--color-secondary', secondary_color)
  root.style.setProperty('--font-heading', font_heading)
  root.style.setProperty('--font-body', font_body)
}

export default function HeroSection({ config, guestName }) {
  const { subject_identity, visual_configs } = config
  const { groom, bride, guest_greeting_template } = subject_identity
  const { moving_text, animated_bg, transition } = visual_configs

    <section className="relative min-h-screen flex flex-col items-center justify-center overflow-hidden">

hooks/useImageCompress.js — Kompresi Otomatis 
export async function compressImage(file, { maxWidth = 1200, quality = 0.82, outputType = 'image/webp' } = {}) {
  return new Promise((resolve) => {
    const img = new Image()
    img.src = URL.createObjectURL(file)
    img.onload = () => {
      const ratio = Math.min(maxWidth / img.width, 1)
      const canvas = document.createElement('canvas')
      canvas.width = img.width * ratio
      canvas.height = img.height * ratio
      canvas.getContext('2d').drawImage(img, 0, 0, canvas.width, canvas.height)
      canvas.toBlob((blob) => resolve(blob), outputType, quality)
    }
  })
}

// Thumbnail 400px untuk grid preview galeri
export async function createThumbnail(file) {
  return compressImage(file, { maxWidth: 400, quality: 0.7 })
}
