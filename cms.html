Integrasi Database Supabase

Untuk menghubungkan sistem Buku Tamu & RSVP secara real-time ke database Anda sendiri, silakan masukkan informasi kredensial Supabase Anda. Jika dikosongkan, sistem otomatis menggunakan **LocalStorage Fallback** (data disimpan lokal di browser).

Skema Tabel Database yang Diperlukan:

Apabila menggunakan Supabase, Anda wajib membuat dua tabel dengan nama berikut:

-- 1. Tabel Buku Tamu (guest_messages)
CREATE TABLE guest_messages (
  id TEXT PRIMARY KEY,
  name TEXT,
  message TEXT,
  created_at TIMESTAMP WITH TIME ZONE
);

-- 2. Tabel RSVP (rsvp_responses)
CREATE TABLE rsvp_responses (
  id TEXT PRIMARY KEY,
  guest_name TEXT,
  attendance TEXT,
  guest_count INTEGER,
  message TEXT,
  created_at TIMESTAMP WITH TIME ZONE
);
